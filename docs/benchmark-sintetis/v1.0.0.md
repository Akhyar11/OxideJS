## Interpretasi

- `training_step` saat ini adalah benchmark sintetis paling siap untuk dijadikan acuan utama karena menghasilkan metrik throughput yang langsung bisa dibandingkan antar versi.
- `transformer_block` berguna sebagai benchmark lapisan bawah untuk melihat dampak optimasi pada attention dan layer normalization secara terpisah.
- `math_synthetic_v1` (pengganti `wikipedia_subset`) sekarang menggunakan dataset matematika. Ini memberikan workload yang lebih stabil dan relevan dengan project generative-bot.

## Tindak Lanjut

- Pertahankan metadata environment lengkap pada snapshot berikutnya agar hasil antar versi tetap comparable.
- Pertahankan benchmark `training_step` sebagai baseline utama lintas versi.
- Gunakan `test/benchmark_sintetis_matematika.ts` sebagai benchmark throughput pada workload teks nyata di masa depan.
