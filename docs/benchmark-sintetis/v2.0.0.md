## Perubahan Penting Dibanding Versi Sebelumnya

- Versi `2.0.0` menandai major update pada stack transformer dibanding seri `1.x`.
- Training transformer sekarang berbasis full-sequence causal LM.
- Backend native sekarang memiliki kernel masked sparse softmax-cross-entropy untuk training dan projector inference khusus LM untuk generation.
- Benchmark inference-only resmi tersedia agar jalur generation dipantau terpisah dari training.

## Interpretasi

- Snapshot `v2.0.0` sebaiknya dianggap baseline utama baru untuk keluarga transformer pada repo ini.
- Seri `1.x` tetap berguna sebagai histori, tetapi tidak lagi setara secara arsitektur untuk dibandingkan langsung tanpa catatan konteks.
- End-to-end transformer throughput pada snapshot ini berada di sekitar `96.94 samples/s` median pada harness pembanding, dengan single-run smoke `npm test` di `64.03 samples/s`.

## Tindak Lanjut

- Jika optimasi lanjutan diperlukan, fokus berikutnya adalah projector full-sequence vocab besar pada training path.
- Pertahankan benchmark inference-only di `transformer_perf_breakdown.ts` agar perubahan generation tidak tertutup oleh noise benchmark training.
- Gunakan `v2.0.0` sebagai baseline baru untuk audit regresi performa transformer.

## Versioning

Versi aktif proyek saat ini adalah `2.0.0`.

Proyek ini memakai format versi `MAJOR.MINOR.PATCH` seperti `2.0.0`.

- Angka paling depan (`MAJOR`): perubahan besar yang biasanya membawa breaking change atau perubahan arsitektur utama.
- Angka tengah (`MINOR`): penambahan fitur baru atau peningkatan yang tetap kompatibel dengan versi sebelumnya.
- Angka paling belakang (`PATCH`): perbaikan bug, optimasi kecil, cleanup, atau perubahan minor yang tidak mengubah API utama.

Contoh:
- `2.0.0`: rilis mayor `2`, minor `0`, patch `0` untuk major update transformer dan backend native.
- `1.1.4`: masih di mayor `1` dan minor `1`, tetapi sudah ada 4 patch/perbaikan kecil dari baseline `1.1.0`.
