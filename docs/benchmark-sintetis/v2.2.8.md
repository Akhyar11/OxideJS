## Perubahan Penting Dibanding Versi Sebelumnya (`v2.2.7`)

1. **Native Sparse Optimizer**: Implementasi native Rust untuk `Adam`, `SGD`, `AdaGrad`, `Momentum`, dan `NAG` yang mendukung update sparse efisien.
2. **Native Sparse Embedding**: Layer `Embedding` kini menggunakan backend Rust untuk akumulasi gradien sparse, menghindari pembuatan matrix gradien besar di sisi JavaScript.
3. **Peningkatan Skalabilitas**: Bottleneck pada vocabulary besar (misal 50k+) telah dieliminasi melalui integrasi native sparse ini.

## Interpretasi

- Versi ini memantapkan layer `Embedding` sebagai layer yang sangat efisien untuk vocab besar.
- Angka benchmark menunjukkan performa yang stabil namun jauh lebih hemat memori karena eliminasi alokasi matrix gradien penuh.
- Peningkatan throughput pada model Transformer terlihat konsisten dengan backend native yang lebih optimal.
