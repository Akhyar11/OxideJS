## Interpretation

- The recurrent family remains the clearest proof that the Rust audit work is paying off; `rnn` exceeded `1200 samples/s` in this synthetic setup.
- `trimPad` remains materially beneficial on the transformer path, with the best throughput among the transformer modes in this snapshot.
- This snapshot should be read together with `docs/correctness/v2.3.1.md` because one earlier combined-suite run in the same session showed a transient recurrent-learning failure before a clean rerun passed.
