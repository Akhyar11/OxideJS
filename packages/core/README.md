# @oxide-js/core

Core tensor, autodiff, optimizer, tokenizer, and native-backend utilities for Oxide-JS.

## Install

```bash
npm install @oxide-js/core
```

## Entry Points

- `Matrix` and `mj` math helpers
- `engine` and `Tape` autodiff
- optimizers and losses
- tokenizer utilities
- Rust native acceleration helpers

See the monorepo documentation:

- Root README: `https://github.com/Akhyar11/Oxide-JS#readme`
- API docs: `https://github.com/Akhyar11/Oxide-JS/tree/main/docs/api`
