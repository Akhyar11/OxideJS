# @oxide-js/layers

Neural-network layers for Oxide-JS, including dense, recurrent, attention, normalization, convolution, and memory-augmented layers.

## Install

```bash
npm install @oxide-js/layers @oxide-js/core
```

## Includes

- `Dense`, `Embedding`, `Dropout`, `Flatten`
- `RNN`, `LSTM`, `GRU`, `AdaptiveMemoryRNN`
- `SelfAttention`, `MultiHeadAttention`, `AttentionPooling`
- `LayerNormalization`, `PositionalEncoding`, `Convolution`, `MemoryBank`

See the monorepo documentation:

- Root README: `https://github.com/Akhyar11/Oxide-JS#readme`
- Layers docs: `https://github.com/Akhyar11/Oxide-JS/tree/main/docs/api`
