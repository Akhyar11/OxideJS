# @oxide-js/models

Model-level APIs for Oxide-JS, including the custom-architecture `Module` system and legacy high-level wrappers.

## Install

```bash
npm install @oxide-js/models @oxide-js/layers @oxide-js/core
```

## Includes

- `Module`, `ModuleList`, `SequentialBlock`
- `Trainer`, `EpisodeTrainer`
- `Sequential`, `RecurrentModel`, `Transformers`
- `DimentionalityReduction`, `Model`

See the monorepo documentation:

- Root README: `https://github.com/Akhyar11/Oxide-JS#readme`
- Models docs: `https://github.com/Akhyar11/Oxide-JS/tree/main/docs/api`
